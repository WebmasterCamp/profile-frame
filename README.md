# Getting Started

Clone the repository

```bash
git clone git@github.com:saenyakorn/profile-frame.git
```

Install dependencies

```bash
pnpm install
```

Start the dev server

```bash
pnpm dev
```

The dev server will be running at [http://localhost:5173](http://localhost:5173)
